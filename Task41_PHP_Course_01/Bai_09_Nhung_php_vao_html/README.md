
### Nhúng php vào trang html

Có thể nhúng php vào nhiều vị trí trong file.

```php
<?php

	// Khởi tạo giá trị biến, mảng, kết nối,...

?>

<!DOCTYPE html>
<html>
	<head>
		<title>PHP basic</title>
	</head>

	<body>
		<?php

		// Đưa ra kết quả + html format
		
		?>

	</body>
</html>

<?php

	// Hủy giá trị biến, mảng, kết nối,...

?>

```
